TripNow 是专为旅行设计的智能记账工具，让团队出行的费用分摊变得简单、透明。

✨ 核心功能：

- 对冲欠款关系，降低转账金额

- 预收公款，统一支付
- 实时余额提醒
- 公账流水清晰可见

👥 多账本管理
- 支持多个旅行账本
- 账本归档/恢复
- 影子成员系统（先记账，后认领）

🎯 适用场景：
- ✈️ 旅行团建
- 🏠 合租生活
- 🎉 聚会活动

- 比手动记账快 3 倍（拍照识别）

立即下载，让旅行记账不再是负担！
