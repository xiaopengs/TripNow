## 🚀 快速提交流程（5步）

### ⏱️ 第1步：准备截图（10分钟）

#### 截图要求
- **数量：** 5张
- **尺寸：** 750 x 1334 px
- **格式：** PNG
- **大小：** < 500KB/张

#### 快速截图方法

```bash

# 2. 打开项目：/Users/ray/.openclaw/workspace/project/TripNow/weixin
# 3. 模拟器设置为 iPhone 6/7/8
# 4. 截图快捷键：Cmd + S
# 5. 按以下顺序截取5张图
```

**截图顺序：**

1. **首页** - 账本列表（空状态或有数据）
2. **记账页** - 填写完整的表单
3. **流水页** - 账单列表（至少3条）
4. **结算页** - 生成结算方案（可空白状态）
5. **公账页** - 公账余额和充值记录

**技巧：**
- 使用开发者工具的"录制"功能快速截取
- 确保界面美观，无错位
- 文字清晰可读

## ⚠️ 注意事项

### 审核可能被拒的原因

1. **功能不完整**
   - ❌ 点击按钮无反应
   - ✅ 预防：所有功能都有响应

2. **类目不符**
   - ❌ 选择错误的类目
   - ✅ 预防：选择"工具 > 记账"

3. **诱导分享**
   - ❌ 强制分享才能使用
   - ✅ 预防：分享功能为可选

4. **虚拟支付**
   - ❌ 在小程序内完成支付
   - ✅ 预防：仅展示收款码

5. **隐私问题**
   - ❌ 缺少隐私政策
   - ✅ 预防：确保链接可访问
