## 欢迎使用 TripNow！

为使用 TripNow 服务（以下简称"本服务"），您应当阅读并遵守《TripNow 用户服务协议》（以下简称"本协议"）。请您务必审慎阅读、充分理解各条款内容，特别是免除或限制责任的相应条款。

**您使用本服务即视为您已阅读并同意上述协议的约束。**

## 第一条 服务内容

### 1.1 服务范围

TripNow 提供以下服务：

1. **账本管理**
   - 创建、编辑、删除账本
   - 账本归档与恢复
   - 多账本管理

2. **记账功能**
   - 手动记账
   - 费用分摊计算
   - 成员管理

3. **结算功能**
   - 智能结算方案生成
   - 收款码展示
   - 结算状态跟踪

4. **公账管理**
   - 公款充值记录
   - 公账支出管理
   - 余额统计

### 1.2 服务变更

我们保留随时修改、中断或终止部分或全部服务的权利，恕不另行通知。

## 第二条 用户账号

### 2.1 账号使用

本服务当前版本为单机应用，无需注册账号：

- 所有数据存储在您的设备本地
- 不需要网络连接即可使用
- 数据不会上传到服务器

### 2.2 数据安全

- 您应妥善保管设备，避免数据泄露
- 建议定期备份重要数据
- 卸载应用将删除所有数据

4. 因设备故障导致的服务异常
5. 因网络原因导致的服务中断
