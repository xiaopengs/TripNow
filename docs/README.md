### 2️⃣ 详细描述

```
TripNow 是专为旅行设计的智能记账工具，让团队出行的费用分摊变得简单、透明。

✨ 核心功能：

📷 拍照识别（开发中）
- 拍摄小票自动识别金额和类目
- 支持语音记账，说出账单信息即可

💰 智能结算
- 最小路径算法，减少转账次数
- 对冲欠款关系，降低转账金额
- 一键生成微信收款码

🏦 公账管理
- 预收公款，统一支付
- 实时余额提醒
- 公账流水清晰可见

👥 多账本管理
- 支持多个旅行账本
- 账本归档/恢复
- 影子成员系统

🎯 适用场景：
- ✈️ 旅行团建
- 🏠 合租生活
- 🎉 聚会活动
- 👫 朋友AA
```

## 🚀 提交步骤

### 快速提交（30分钟）

详见：[QUICK_SUBMISSION_GUIDE.md](./QUICK_SUBMISSION_GUIDE.md)

### 详细步骤

1. **准备截图**（10分钟）
   - 使用微信开发者工具截图
   - 确保尺寸750x1334px
   - 文件名按规范命名

2. **部署文档**（5分钟）
   - 上传隐私政策到服务器
   - 上传用户协议到服务器
   - 确保链接可访问

3. **编译小程序**（5分钟）
   ```bash
   cd weixin
   npm install
   npm run build:weapp
   ```

4. **上传代码**（5分钟）
   - 使用微信开发者工具上传
   - 版本号：1.0.0
   - 备注：初始版本

5. **提交审核**（5分钟）
   - 登录微信公众平台
   - 填写应用信息
   - 上传截图
   - 提交等待审核
