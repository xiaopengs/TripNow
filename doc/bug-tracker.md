## 📝 问题记录模板

添加新问题时使用以下格式：

```markdown
### BUG-XXX: 问题标题

**基本信息**
- 发现日期：YYYY-MM-DD
- 发现人：xxx
- 模块：FAB / Inbox / 记账 / 首页 / 流水 / 结算
- 优先级：P0 / P1 / P2
- 状态：待修复 / 修复中 / 已修复 / 已关闭

**问题描述**
简明描述问题现象

**复现步骤**
1. 步骤 1
2. 步骤 2
3. 步骤 3

**预期结果**
应该发生什么

**实际结果**
实际发生了什么

**截图/录屏**
如有，添加链接

**环境信息**
- 设备：iPhone 14 / Chrome 120
- 版本：commit xxx
- 网络：WiFi / 4G

**修复记录**
- 修复人：xxx
- 修复日期：YYYY-MM-DD
- 修复方案：简述修复方法
- 相关提交：commit xxx

**验证记录**
- 验证人：xxx
- 验证日期：YYYY-MM-DD
- 验证结果：通过 / 不通过
```
